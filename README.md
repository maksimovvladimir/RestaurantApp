Что это
Программа для управления рестораном: официанты принимают заказы, администратор ведёт меню и склад, клиенты бронируют столики

Как устроено
RestaurantApp.Data — база данных и вся логика работы с ней (PostgreSQL + Npgsql). Основная логика живёт прямо в БД — хранимые процедуры, триггеры

RestaurantApp.UI — окна и кнопки (Windows Forms, .NET 8, только под Windows)

Как запустить
Создать базу: выполнить в терминале psql -f restaurant_db.sql. Если нужны тестовые данные — следом restaurant_seed.sql

Настроить подключение: в файле RestaurantApp.Data/DbConfig.cs прописать свою строку подключения к БД. Можно не трогать код, а задать переменную окружения RESTAURANT_DB_CONNECTION

Собрать и запустить: открыть RestaurantApp.sln в Visual Studio, нажать «Собрать» или выполнить dotnet build, затем запустить проект RestaurantApp.UI

Как добавить тестового пользователя
Либо через код: UsersRepository.CreateUser(...), либо SQL-вставкой напрямую в таблицу users. Пароль хешируется через SHA-256, хеширование описано в UsersRepository.HashPassword
